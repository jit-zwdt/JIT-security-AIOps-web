Linux

  1. 修改中间件tomcat安装目录下的catalina.sh文件

    添加以下内容 
    CATALINA_OPTS="-Dcom.sun.management.jmxremote -Dcom.sun.management.jmxremote.authenticate=false -Dcom.sun.management.jmxremote.ssl=false -Djava.rmi.server.hostname=*.*.*.*"
    注：hostname=TOMCAT服务器IP地址，该IP必须能与ZABBIX Server能通信，某些情况下IP经过映射转换，那么需要使用转换后的地址。
  2. 修改中间件tomcat安装目录下tomcat server.xml 配置文件
    # vi /usr/local/tomcat/conf/server.xml
    添加以下内容:
    <Listener className="org.apache.catalina.mbeans.JmxRemoteLifecycleListener" rmiRegistryPortPlatform="12345" rmiServerPortPlatform="12346" />
    注：这里的端口号12345和12346可以自定义，如果同一台机器上有运行多个Tomcat程序，每个tomcat的监控操作步骤都一样，但是这里的监听端口每个Tomcat程序必须不同，否则端口会冲突
    如果上述命令无法打开JMV端口则尝试修改catalina.sh文件：
    CATALINA_OPTS="${CATALINA_OPTS}  -Djava.rmi.server=IP -Djavax.management.builder.initial= -Dcom.sun.management.jmxremote=true -Dcom.sun.management.jmxremote.port=12345 -Dcom.sun.management.jmxremote.ssl=false -Dcom.sun.management.jmxremote.authenticate=false"
  3. 重启tomcat服务
    # cd /tomcat/apache-tomcat-6.0.41/bin/
# ./shutdown.sh
# ./startup.sh
  4. 查看端口是否监听 
    # netstat –nlp | grep 12345
# netstat –nlp | grep 12346
  5. 开放防火墙端口
    # vim /etc/sysconfig/iptables -A INPUT -m state --state NEW -m tcp -p tcp --dport 端口 -j ACCEPT
  6. 重启防火墙
    # service iptables restart

Window

  1. 修改中间件tomcat的安装目录下的catalina.bat文件
    添加以下内容：
    set CATALINA_OPTS=-Dcom.sun.management.jmxremote -Dcom.sun.management.jmxremote.authenticate=false -Dcom.sun.management.jmxremote.ssl=false -Djava.rmi.server.hostname=*.*.*.* -Dcom.sun.management.jmxremote.port=12345
    注： hostname=TOMCAT服务器IP地址，该IP必须能与ZABBIX Server能通信，某些情况下IP经过映射转换，那么需要使用转换后的地址；port为jmx监控端口 这里的端口号12345可以自定义，如果同一台机器上有运行多个Tomcat程序，每个tomcat的监控操作步骤都一样，但是这里的监听端口每个Tomcat程序必须不同，否则端口会冲突
  2. 重启tomcat服务 
    >shutdown.bat
>startup.bat
  3. 查看端口是否监听
    > netstat -ano| findstr "12345"
  4. 开放防火墙端口
    如果系统防火墙处于启动状态，则需要开放jmx监控端口，才能正常监控。
    注：如果您通过 Apache Tomcat Windows Service Installer（而不是向 CATALINA_OPTS 添加选项）安装了 Apache Tomcat，那么请运行”/bin/Tomcat6w.exe”或”/bin/Tomcat7w.exe”，并在“属性”窗口的 Java 选项卡中添加以下值:
    -Dcom.sun.management.jmxremote.port=12345
-Dcom.sun.management.jmxremote.authenticate=false
-Dcom.sun.management.jmxremote.ssl=false
  5. 若提示没有安装服务，则运行以下命令为Tomcat服务开启Apache服务管理功能：
    >0 Tomcat6w.exe //MS//Tomcat服务名
